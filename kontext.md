# Kontext pro redesign webu Studio Hanky Víchové

## 1. Základní informace o projektu

Webová stránka existuje na adrese: https://www.pedikura-havirov.cz/

Cílem je vytvořit novou moderní webovou prezentaci ve formátu HTML/CSS/JavaScript. Současný web působí zastarale, obsah je roztříštěný a některé informace nejsou dostatečně přehledné. Nový web má působit profesionálně, důvěryhodně, příjemně a lidsky. Má jasně ukázat, že Studio Hanky Víchové není jen klasická pedikúra/manikúra, ale místo, kde se propojuje péče o krásu, zdraví, pohodlí klienta a dlouholetá zkušenost.

Web bude určen hlavně pro klienty z Havířova a okolí, kteří hledají pedikúru, manikúru, péči o problematická chodidla nebo masáže. Důležitá cílová skupina jsou také diabetici a lidé, kteří řeší zarůstající nehty, kuří oka, otlaky, vbočené palce, kladívkové prsty nebo bolest chodidel.

Komunikace webu má být v češtině a ve vykání.

## 2. Hlavní cíle redesignu

- zpřehlednit služby a ceník,
- posílit důvěryhodnost studia,
- zdůraznit dlouholetou praxi Hanky Víchové od roku 1995,
- jasně komunikovat zdravotně orientovanou pedikúru a práci s problematickými chodidly,
- upozornit na možnost ošetření diabetiků a možnost čerpání příspěvku od některých zdravotních pojišťoven,
- zviditelnit možnost online rezervace,
- zlepšit mobilní použitelnost,
- vytvořit modernější, jemnější a estetičtější vizuální styl,
- sjednotit strukturu a odstranit duplicity v navigaci.

## 3. Doporučené technologické zadání

Vytvořit statický web v:

- HTML5,
- CSS3,
- čistý JavaScript bez složitého frameworku,
- responzivní layout pro mobil, tablet i desktop,
- důraz na rychlost načítání,
- čistý a přehledný kód,
- SEO-friendly struktura nadpisů,
- přístupnost: dostatečný kontrast, čitelné fonty, jasné CTA, alt texty u obrázků.

Doporučuji vytvořit web jako jednostránkový landing page s kotvami v navigaci. Pokud bude potřeba více prostoru, lze služby rozdělit na podstránky, ale preferovaná varianta je přehledný one-page web.

## 5. Vizuální styl

Web by měl působit:

- čistě,
- jemně,
- profesionálně,
- důvěryhodně,
- lidsky,
- klidně,
- pečujícím dojmem.

Doporučená barevnost:

- teplá béžová / krémová jako pozadí,
- jemná tělová růžová nebo pudrová jako akcent,
- světle hnědá nebo taupe pro doplňky,
- tmavší hnědošedá pro text,
- bílá pro karty a čisté plochy.

Vyhnout se příliš ostrým barvám, křiklavé růžové a generickému salonnímu vzhledu.

Doporučené fonty:

- nadpisy: elegantní serif nebo jemný moderní font,
- texty: dobře čitelný sans-serif.

Příklad kombinace:

- Playfair Display / Cormorant Garamond pro nadpisy,
- Inter / Lato / Nunito Sans pro běžný text.

## 6. Fotografie a grafika

Použít ideálně skutečné fotografie studia, týmu a detailů péče. Pokud nebudou k dispozici, připravit layout tak, aby šly fotografie jednoduše doplnit.

Doporučené typy fotografií:

- detail čistého pracovního prostředí,
- ruce při péči,

- kosmetické/přírodní produkty,
- portréty členek týmu,
- detail interiéru studia.

Nepoužívat laciné stock fotografie s přehnaně umělým beauty stylem.

Doporučené ikonky:

- diabetici,
